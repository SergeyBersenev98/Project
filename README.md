## Setup

To run this project, install it localy using npm:

```shell
$ git clone https://github.com/SergeyBersenev98/SergeyBersenev98.github.io
$ cd ./SergeyBersenev98.github.io
$ npm install
$ npm start
```

Then open your browser on http://localhost:3000/
